Тестовый коммит для проверки workflow с артефактами

Это изменение предназначено исключительно для тестирования новой функциональности загрузки артефактов в PR комментарии.

После merge этого PR в workflow Runa должен:
1. Создать артефакт (test-artifact.txt)
2. Загрузить его в репозиторий
3. Отобразить как встроенный файл в комментарии к PR

Результат можно будет увидеть в комментариях к этому PR.
